# Support

## Getting Help

If you need help with Guardian Framework, here are the best ways to get it:

### 📖 Documentation

- **[README.md](README.md)** — Overview, quick start, CLI reference
- **[docs/](docs/)** — Architecture guide, user guide, and design documentation
- **[WORKFLOW.md](WORKFLOW.md)** — Agent workflow quick reference
- **[CONTRIBUTING.md](CONTRIBUTING.md)** — Development guide for contributors

### 🐛 Bug Reports & Feature Requests

- **GitHub Issues:** [Create a new issue](https://github.com/arman-jalili/guardian-framework/issues/new/choose)
- Use the [bug report](https://github.com/arman-jalili/guardian-framework/issues/new?template=bug_report.yml) or [feature request](https://github.com/arman-jalili/guardian-framework/issues/new?template=feature_request.yml) templates
- Include your Bun version, OS, and Guardian version (`guardian-framework --version`)

### 💬 Discussions

- **GitHub Discussions:** [Start a discussion](https://github.com/arman-jalili/guardian-framework/discussions)
- **Discord:** _(coming soon)_

### 🔒 Security Issues

For security vulnerabilities, **do not** open a public issue. See [SECURITY.md](SECURITY.md) for our responsible disclosure policy.

### ⏱ Response Times

| Issue Type | Initial Response | Resolution Target |
|-----------|-----------------|-------------------|
| Security vulnerability | < 48 hours | < 7 days |
| Bug (critical) | < 72 hours | < 14 days |
| Bug (non-critical) | < 1 week | < 30 days |
| Feature request | < 1 week | Next milestone |
| General question | < 3 days | — |
